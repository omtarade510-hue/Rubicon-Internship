## 🚀 Getting Started

### Prerequisites

* Install Python 3 on your system

### How to Run

1. Clone the repository:

```bash
git clone https://github.com/omtarade510-hue/Rubicon-Internship.git
```

2. Navigate to the project folder:

```bash
cd Rubicon-Internship
```

3. Run any Python file:

```bash
python filename.py
```
